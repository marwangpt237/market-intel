## How to Add a New Collector

```python
from collectors.marketplace.base import MarketplaceCollector, CollectorMetadata
from core.models import RawItem

class MyCollector(MarketplaceCollector):
    metadata = CollectorMetadata(
        name='my_source',
        country='DZ',                # ISO code
        category='classifieds',      # news, classifieds, jobs, forum, government, social
        entity_types=['product'],    # what entities this produces
        description='My custom data source',
        rate_limit_per_hour=60,
        reliability=0.75,            # 0-1, used by TrustLayer
        cost_per_call=0.0,
        required_credentials=['MY_API_KEY'],  # env var names
        tags=['algeria', 'custom'],
    )

    def collect(self) -> list[RawItem]:
        # Your collection logic here
        return [RawItem.create(
            source='my_source',
            source_name='My Source',
            title='...',
            url='...',
            body='...',
        )]

# Register it
from collectors.marketplace.base import CollectorRegistry
CollectorRegistry.register(MyCollector())
```

The collector will then be:
- Discoverable via `CollectorRegistry.list_by_country/category/entity_type`
- Health-monitored by `CollectorHealthMonitor`
- Trust-weighted by `TrustLayer` (using `metadata.reliability`)
- Listed in this report automatically
